FAQ +

¿Dónde realizo el pago de mis tickets?

Una vez introduce los datos en el formulario superior, se enviará al correo electrónico y número de télefono (si se indica) proporcionado un enlace con la página para realizar el pago de forma segura.

¿De que formas puedo realizar el pago?

Para realizar el pago se tiene que hacer via internet, desde el enlace proporcionado.
Se facilita el pago mediante tarjeta de crédito, PayPal, transferencia o pagos desde el móvil.
